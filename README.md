### Getting Started

This project uses [CocoaPods](http://cocoapods.org/).

``` bash
$ git clone https://github.com/danylokostyshyn/popcorntime-ios.git
$ cd popcorntime-ios/
$ pod install
$ open PopcornTime.xcworkspace/
```
